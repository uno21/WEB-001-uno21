Ofrecemos un curso integral, agnóstico del lenguaje de programación, que le permitirá sentar las bases de diseños reusables, extensibles, mantenibles, flexibles y por sobre todas las cosas, perdurables en el tiempo.

Programación Orientada a objetos

Descripción

Un curso agnóstico del lenguaje de programación, cuyo ojetivo es formar profesionales que puedan entender, comunicar y construir soluciones software utilizando la programación orientada a objetos, y empleando las mejores prácticas del mercado.

Se abarcan unidades que incluyen tanto las primeras nociones como conceptos y principios avanzados que destacarán a sus profesionales.

A quiénes está dirigido

Este curso está dirigido a desarrolladores que dominan otros paradigmas y necesiten sumergirse en el desarrollo orientado a objetos, así como también nuevas incorporaciones que necesiten capacitación.

Duración

32 horas

Alternativas

Puede dictarse una capacitación más breve (16 horas) que abarque los conceptos básicos de programación orientada a objetos

Puede agregarse un módulo (+8 horas) de taller en un lenguaje de programación específico

Programación en Java

Descripción

Nuestro conjunto de cursos de Java está orientado a conocer, practicar y llegar a manejar la tecnología en forma autónoma.

Para ello ofrecemos una orientación en cuanto a los cursos a tomar, y la profundidad de los mismos. Del mismo modo, ajustamos la práctica según las necesidades del grupo

A quiénes está dirigido

Este curso está dirigido desarrolladores y no desarrolladores que deseen conocer la tecnología, o que ya habiendo dado los primeros pasos tengan interés en dominar aspectos más complejos del lenguaje, o capacidades más avanzadas del mismo

Cursos ofrecidos

  • Introducción al lenguaje Java (32 horas)

    Uso de variables, objetos, estructuras de control de flujo, métodos y atributos, encapsulamiento, constructores, tipos de dato atómicos y estructurados, herencia básica

  • Programación básica en Java (40 horas)

    Identificadores, tipos, expresiones, control de flujo, arrays, diseño de clases, excepciones, framework de colecciones de java (JCF), patrones de diseño, threads

  • Java Enterprise Edition (40 horas)

    Model-view-controller (MVC), Tests unitarios, JDBC, diseño en capas, RMI, Java web

  • Spring (24 horas)

    Introducción a Spring, conceptos básicos, inyección de dependencias, integración con otros módulos, SpringMVC, anotaciones, vistas

  • Hibernate (16 horas)

    Conceptos básicos, qué es un ORM, configuración de Hibernate, Session Factory, mapeo de clases con Annotations, consultas con Criteria, relaciones 1 a 1, 1 a n, n a n, herencia

  • Hibernate + Spring (32 horas)

    Incluye ambos cursos, combinados para llevar adelante una mejor solución

Bases de Datos

Descripción

A quiénes está dirigido

Este curso está dirigido a desarrolladores que dominan otros paradigmas y necesiten sumergirse en el desarrollo orientado a objetos, así como también nuevas incorporaciones que necesiten capacitación.

Cursos ofrecidos

  • Introducción a las bases de datos (8 horas)

    Conceptos de bases de datos, tipos. Características, utilización y potencialidades

  • Programación SQL (40 horas)

    Creación de esquemas, consultas, vistas, triggers, procedimientos almacenados, funciones. Incluye un taller integrador

  • Introducción a las bases de datos no relacionales (8 horas)

    Conceptos de bases de datos, tipos. Características, utilización y potencialidades. Ejemplos y alcance

  • Aprendiendo MongoDB (32 horas)

    Introducción. Conceptos básicos. CRUD con MongoDB. Consultas. Índices, agregación, ejemplos. Conceptos avanzados (replica y sharding)
